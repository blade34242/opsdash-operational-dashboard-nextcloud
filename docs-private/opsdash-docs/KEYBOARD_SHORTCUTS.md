# Keyboard Shortcuts

Shortcuts are defined in `opsdash/src/services/shortcuts.ts` and rendered by `KeyboardShortcutsModal.vue`.
Keep this document minimal and update only if the list changes.

Current groups:
- Navigation (Alt + arrows, Alt + Shift + R)
- Notes (Ctrl/Cmd + S)
- Targets (Alt + T)
- Misc (? / Shift + C / Shift + E)
